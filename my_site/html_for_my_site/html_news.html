Компания Google продолжает экспериментировать с дизайном популярного видеохостинга YouTube. В уже сети появились первые скриншоты нового пользовательского интерфейса сервиса на Android-устройствах — в дальнейшем же, по данным сетевых информаторов, изменения ожидают пользователей десктопов и умных телевизоров. YouTube Согласно утечке с закрытого бета-теста новой версии приложения YouTube для Android, одним из главных нововведений интерфейса станет панель поиска. На смену привычной иконке в форме увеличительного стекла придёт поисковая строка в браузерной стилистике, которая заменит привычный логотип сервиса в верхнем левом углу стартовой страницы. Другое изменение касается взаимодействия пользователя с его учётной записью. При запуске приложения без авторизации на экране будет появляться всплывающее окно с предложением вход через учётную запись Google. Инсайдеры предполагают, что такая «мотивация» позволит YouTube получать больше статистики для показа контекстной рекламы — впрочем, функция носит рекомендательный характер и не влияет на работу приложения. Сейчас компанией проводится закрытое тестирование нового интерфейса — большинству пользователей изменения ещё недоступны. Дата возможного апдейта дизайна YouTube на поддерживаемых платформах источником не сообщается. Источник: androidcentral.com
